Deep Learning for Fashion Image Classification: A Comparative Study of CNN and Feedforward Networks

Jan 2023 - Mar 2023 · Tecnologico de Monterrey

1. Project Scope

This project involved the supervised image classification of clothing items from the Fashion MNIST dataset using two deep learning models: a Convolutional Neural Network (CNN) and a Fully Connected Feedforward Neural Network. The goal was to compare their performance across architectural complexity, accuracy, generalization, and computational efficiency.

2. CNN Architecture and Training

The CNN model was implemented using the TensorFlow Keras Sequential API. The architecture consisted of convolutional layers with ReLU activation, max pooling layers for spatial downsampling, dropout layers for regularization, and dense layers for classification. The final output layer used softmax activation for multiclass prediction.

Data was normalized to [0,1] and reshaped for compatibility. A learning rate scheduler dynamically adjusted the learning rate post-epoch 50. The model was trained for 100 epochs using the Adam optimizer and categorical crossentropy as the loss function. It achieved a final test accuracy of 92.1%, demonstrating high generalization with minimal overfitting.

3. Feedforward Neural Network ("TrendyNet")

The alternative model was a deep feedforward neural network comprising multiple dense layers with ReLU activations and dropout. Input images were flattened into 1D vectors before being fed into the network. The model was compiled and trained under similar hyperparameter conditions.

While this architecture is less computationally demanding, it achieved a lower test accuracy of 87.8%. Validation curves showed higher variance compared to CNN, indicating reduced generalization under complex data patterns.

4. Evaluation and Metrics

Both models were evaluated using classification accuracy and confusion matrices. The CNN consistently outperformed the feedforward model across all metrics, particularly on visually complex classes such as “Shirt” and “Pullover”. CNN also showed more stable loss and accuracy curves, indicating better convergence.

5. Computational Efficiency

Training time for the CNN model was approximately 10 minutes and 42 seconds, while the feedforward model required 3 minutes and 39 seconds. The trade-off between computational cost and classification performance was highlighted, with CNN offering superior accuracy at the expense of processing time and resource usage.

6. Optimization Opportunities

Improvements for the CNN model include parameter pruning, model compression, and exploration of pretrained networks like ResNet or MobileNet. For the feedforward model, further hyperparameter tuning and feature extraction techniques could help close the performance gap.

7. Technical Takeaways

The project reinforced key practices in deep learning, including:

  • Appropriate architecture selection for image data
  • Use of dropout and L2 regularization to prevent overfitting
  • Importance of data preprocessing and normalization
  • Learning rate scheduling for optimizing convergence

Natural Language Classifier for Operational Fault Reports in Industrial Environments (NLC Ternium)

Jan 2023 - Aug 2023 · Tecnologico de Monterrey

1. Project Scope

The project aimed to automate the classification of industrial maintenance records by applying supervised machine learning techniques to unstructured textual descriptions. The objective was to generate structured labels ("Symptom" and "Cause") from free-text fields within Ternium’s fault report database.

2. Data Preparation

A raw dataset containing over 64,000 entries was cleaned and standardized. Preprocessing steps included conversion to lowercase, removal of accents and punctuation, elimination of stopwords using NLTK, and duplicate record removal. Categorical variables were encoded numerically, and key input fields were tokenized to enable effective feature extraction.

3. Model Pipeline and Training

TF-IDF vectorization was applied to extract numerical features from text data. Multiple classification models were trained using Scikit-learn pipelines, including Random Forest, Naive Bayes, K-Nearest Neighbors (KNN), SVM, and Logistic Regression. Hyperparameters were optimized using GridSearchCV, and the best-performing model was selected based on accuracy metrics.

4. Results

The Random Forest classifier yielded the highest accuracy at 61% when using concatenated input variables. Model evaluation showed improved prediction consistency when combining multiple text fields, confirming the value of contextual feature enrichment.

5. Operational Impact

The classification system enables faster interpretation of failure reports and reduces manual effort in data labeling. It supports preventive maintenance planning and contributes to better fault traceability and decision-making within industrial operations.

6. SDG Alignment

The solution supports Sustainable Development Goal 9 (Industry, Innovation and Infrastructure) through digital transformation, and SDG 12 (Responsible Consumption and Production) by optimizing data use and reducing production downtime.
